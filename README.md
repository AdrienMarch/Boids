# Boids Simulation (Work In Progress)

This project contains an interactive Boids flocking simulation built in a Jupyter notebook.

## Status

Work in progress. The model, parameters, and notebook structure may change frequently.

## Project Structure

- `Boids.ipynb`: main notebook containing the simulation code and animation.
- `README.md`: project notes and setup instructions.

## Notes

- Expect temporary code and experimental parameters.
- Results and visuals can change as rules and constants are tuned.
